# Open chat

Open chat é uma aplicação web cujo o principal objectivo é a troca de informações (mensagens) rapidamente
e com uma privacidade extrema.

### Objectivos

- Troca de informações individuais
- Privacidade

### Ferramentas

- Python
- Flask
- Pydantic
- Mongo DB
- Html
- Css
- Javascript
- Firebase
